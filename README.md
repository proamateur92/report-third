# 항해 99 리액트 3주차 과제
- redux를 사용하여 상태를 관리한다.
- firestore를 통해 redux상태와 연동하였다.
- firestore로 데이터 호출, 등록, 수정, 삭제 기능을 구현하였다.
- firebase storage를 사용하여 이미지 업로드 기능을 구현하였다.
